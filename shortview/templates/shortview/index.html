{% trans "What is ShortView?" %}

{% trans "With this platform, you can shorten a link you want to share with people, and monitor when these links have been opened." %} {% trans "So it allows do do two things at the same time: shorten a long link if you need to, and allow tracking of the link!" %}

{% trans "How does this work?" %}

{% trans "To achieve that, this platform will create a shortened link to the destination url you want to share." %} {% trans "When people will click this link, they will send a request to our servers with the id of the link they clicked." %} {% trans "The server will then register that this link has been clicked, with data such as the ip of the person who clicked and the date." %} {% trans "After registering the action, the server will perform a redirection to the destination url." %} {% trans "This whole process is practically instant for the user who clicked the link." %}

{% trans "Who can I send tracked links to?" %}

{% trans "Make sure that the people you're going to send tracked links to are aware that their clicks will be registered." %} {% trans "Never send it without telling that you're collecting the person's activity, as it may constitute a privacy violation." %}
